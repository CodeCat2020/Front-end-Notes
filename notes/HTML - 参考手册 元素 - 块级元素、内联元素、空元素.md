

# 块级元素

块级元素一定不是空元素？不是，比如 **`<hr>`**。

| 块级标签                |                                    |
| ----------------------- | ---------------------------------- |
| **`<address>`**         | 联系方式信息。                     |
| `<article>` （HTML5）   | 文章内容。                         |
| `<aside>` （HTML5）     | 伴随内容。                         |
| `<audio>` （HTML5）     | 音频播放。                         |
| `<blockquote>`          | 块引用。                           |
| `<canvas>`（HTML5）     | 绘制图形。                         |
| `<dd>`                  | 定义列表中定义条目描述。           |
| `<div>`                 | **文档分区**。                     |
| `<dl>`                  | 定义列表。                         |
| `<fieldset>`            | 表单元素分组。                     |
| `<figcaption>`（HTML5） | 图文信息组标题                     |
| `<figure>`（HTML5）     | 图文信息组 (参照 `<figcaption>`    |
| `<footer>`（HTML5）     | 区段尾或页尾。                     |
| **`<form>`**            | 表单。                             |
| **`<h1>`-`<h6>`**       | 标题级别 1-6.                      |
| `<header>`（HTML5）     | 区段头或页头。                     |
| `<hgroup>`（HTML5）     | 标题组。                           |
| **`<hr>`**              | 水平分割线。                       |
| `<li>`                  |                                    |
| `<noscript>`            | 不支持脚本或禁用脚本时显示的内容。 |
| **`<ol>`**              | 有序列表。                         |
| `<output>`（HTML5）     | 表单输出。                         |
| **`<p>`**               | 行。                               |
| `<pre>`                 | 预格式化文本。                     |
| `<section>` （HTML5）   | 一个页面区段。                     |
| **`<table>`**           | 表格。                             |
| `<tfoot>`               | 表脚注。                           |
| **`<ul>`**              | 无序列表。                         |
| `<video>`（HTML5）      | 视频。                             |
